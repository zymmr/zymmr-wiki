Search Work Item in Zymmr

Overview​

The Search Work Item feature is integrated throughout the Zymmr product, allowing users to quickly find work items using the key or title. This functionality ensures efficient navigation and task management.

How to Search for a Work Item​

Access the Search Bar​

  • Navigate to any page within Zymmr where the search bar is available.

Enter Search Criteria​

  • Key: Enter the unique key of the work item you are looking for.
  • Title: Enter the title or part of the title of the work item.

View Search Results​

  • The search results will display matching work items based on the key or title entered.
  • Click on the desired work item from the search results to view its detailed information.
note
  • The search functionality is available on various pages, ensuring you can search for work items from anywhere within the Zymmr product.
  • Using the work item key provides a precise search result, while using the title allows for broader searches.
