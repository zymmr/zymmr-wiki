Releases

What is a Release?​

A release in Zymmr is a specific version or segment of a project that is delivered to stakeholders. Releases help manage and track project deliverables, ensuring each version is documented and delivered on schedule.

How to Create a Release in Zymmr​

Navigate to Project Listing Page​

  1. From the Projects listing, search for the project you want to configure.
  2. Click on the project to open it.
  3. On the right side, click on the 'Project Configuration Settings' icon.

Access the Releases Section​

  1. In the configuration settings, look for the Releases section.
  2. Click on the Releases section to open it.

Create a Release​

  1. Click on the 'Create Release' icon.
  2. Fill in the following details:
    • Release Name: Enter a descriptive name for the release.
    • Start Date: Define the start date of the release.
    • End Date: Define the end date of the release.
    • Description: Provide a detailed description of the release.
  3. Click the 'Save' button to save the details.
  4. Once saved, all releases are listed.

View and Manage Releases​

All releases for the project will be listed with the following information:

  • Release Name
  • Start Date
  • End Date
  • Status: Shows if the release is unreleased or has been released.

Options available for each release include:​

  • Release: Release this changing the status to ‘Released’

  • Archive: Archive the release.

  • Edit: Edit the release details.

  • Delete: Delete the release.

  • Note: All releases are considered unreleased until the status is changed to "Released."

  • The release listing also includes a search function to find specific releases.

  • Note: If a work item is associated with a release, the release cannot be deleted. An error message will indicate this restriction.

How to Search and Edit a Release​

Search for a Release​

  1. Use the search function in the Releases section to find the release you want to edit.

Edit Release Details​

  1. Click on the 'Edit' icon in the options next to the release you want to modify.
  2. Update the necessary details such as Release Name, Start Date, End Date, and Description.
  3. Click 'Save' to apply the changes.

How to Delete a Release​

Search for the Release​

  1. Use the search function in the Releases section to find the release you wish to delete.

Delete the Release​

  1. Click on the 'Delete' icon next to the release.
  2. Confirm the deletion when prompted.
    • Note: If a work item is associated with a release, the release cannot be deleted. An error message will indicate this restriction.

Important Note​

  • Releases will remain in the "unreleased" state until the status is explicitly changed to "released," impacting the project's tracking and reporting.
