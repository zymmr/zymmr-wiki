Milestones in Zymmr

What is a Milestone?​

A milestone is a significant event or a marker in a project timeline that signifies the completion of a major phase or work item. In Zymmr, milestones are defined at a project level and help in tracking progress, ensuring that the project is on schedule. They can be associated with specific work items.

How to Create a Milestone in Zymmr​

Navigate to Project Configuration Settings​

  1. From the Projects listing, search for the project you want to configure.
  2. Click on the project to open it.
  3. On the right side, click on the 'Project Configuration Settings' icon.

Choose Milestone​

  1. In the configuration settings, look for the Milestone section. The icon is represented by four horizontal lines.
  2. Click on the Milestone section to open it.

Create a Milestone​

  1. Click on the 'Create Milestone' icon.
  2. Fill in the following details:
    • Milestone Name: Enter a descriptive name for the milestone.
    • Milestone Type: Select the type of milestone (Engineering or Billing).
    • Owner: Assign a user responsible for the milestone.
    • Status: Set the status of the milestone (In Progress, Completed).
    • From Date: Define the start date of the milestone.
    • To Date: Define the end date of the milestone.
  3. Click 'Save' to create the milestone.

How to Search and Edit a Milestone​

Search for a Milestone​

  1. In the Milestone section of the Project Configuration Settings, use the search bar to find the milestone you want to edit.

Edit Milestone Details​

  1. Click on the milestone from the search results.
  2. Update the necessary details such as Milestone Name, Type, Owner, Status, From Date, and To Date.
  3. Click 'Save' to apply the changes.

How to Delete a Milestone​

Search for the Milestone​

  1. Use the search function in the Milestone section to find the milestone you wish to delete.

Delete the Milestone​

  1. Select the milestone from the search results.
  2. Click on the 'Delete' icon.
  3. Confirm the deletion when prompted.

By following these steps, you can effectively manage milestones in Zymmr, ensuring that your project timelines are well-organized and transparent.

Limitation​

  • Currently, milestones are not visible across the Gantt chart to provide a clear visual representation of project timelines.
