Active Sprint Board in Zymmr

Overview​

The Active Sprint Board in Zymmr is a visual tool designed to track the progress of work items within an active sprint. It provides a clear view of work items at various stages, aiding effective sprint management.

Default Columns on the Active Sprint Board​

  • Open: Displays newly added work items that have not yet started.
  • Approved: Contains work items that have been reviewed and are ready to be worked on.
  • In Progress: Shows work items actively being worked on.
  • Done: Lists completed work items that are ready for review or closure.

How to Access the Active Sprint Board​

  • Navigate to the Active Sprint Board:
    Go to the Project Backlog page.
  • Use the Active Sprint option on the left to access the Active Sprint Board.

How to Use the Active Sprint Board​

Viewing Work Items​

View work items in columns labeled Open, Approved, In Progress, and Done.

Managing Work Items​

  • Move Work Items: Drag and drop work items between columns to update their status.
  • Create New Work Items: Click the Create button to add work items to the Open column.
  • Update Work Item Status: Drag work items to the appropriate column as they progress.

Tracking Progress​

Monitor work item flow to assess sprint progress and identify bottlenecks.

Customizing the Active Sprint Board​

Customizing Columns​

  • Columns can be customized using the Board Settings option.
  • Access Board Settings by clicking the settings icon on the right (represented by {...}).
  • For more details, refer to the Board Settings Documentation.

Customizing Card Items​

  • Customize the items displayed on a card using Board Customize.
  • Access Board Customize by clicking the customization icon on the right (represented by {...}).
  • For more details, refer to the Board Customize Documentation.

Workflow Status Definitions​

The workflow status definition supports the specification of both the start and end statuses of a workflow. For more information, refer to the Workflow Management Documentation.

Note

  • The columns Open, Approved, In Progress, and Done can be customized to suit your needs.
  • Ensure work items are correctly placed in columns to maintain accurate progress tracking.

Limitation​

The default columns and their labels cannot be altered without administrative permissions. For specific needs, consult with the system administrator.
